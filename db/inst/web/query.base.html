Query database

Usage

query.base(query, con = NULL, ...)

Arguments

query
SQL query string
con
database connection object
...
optional arguments for connecting to database (e.g. password, user name, database)

Value

data frame with query results

Description

Generic function to query trait database

Details

Given a connection and a query, will return a query as a data frame

Deprecated, please use db.query

Examples

## <strong>Not run</strong>: 
# query.base('select count(id) from traits;')
